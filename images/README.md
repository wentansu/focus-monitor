# Heart Rate Visualization Images

This folder contains all heart rate visualization images generated by the Solace orchestrator agent.

## File Naming Convention

- **Timestamped files**: `heart_rate_analysis_YYYYMMDD_HHMMSS.png`
  - Permanent copies with creation timestamp
  - Useful for tracking trends over time

- **Latest files**: `heart_rate_analysis_latest.png`
  - Always points to the most recent visualization
  - Convenient for quick access or embedding in dashboards

## Image Types

### 1. Heart Rate Distribution (`heart_rate_distribution_*.png`)
- Histogram showing the frequency of different BPM ranges
- Helps identify patterns and most common heart rates

### 2. Heart Rate Analysis (`heart_rate_analysis_*.png`)
- Line chart showing BPM over time in the monitoring session
- Includes average BPM reference line
- Shows trends and variations

## How Images Are Generated

1. Heart rate monitor collects 10 BPM readings
2. Data is sent to the Solace orchestrator agent
3. Orchestrator analyzes the data and creates visualizations
4. The `monitor_artifacts.py` script automatically copies them here

## Viewing Images

- **macOS**: `open images/heart_rate_analysis_latest.png`
- **Linux**: `xdg-open images/heart_rate_analysis_latest.png`
- **Any OS**: Just double-click the .png files in your file explorer

## Usage in Applications

These images can be used for:
- Health dashboards
- Progress tracking
- Medical reports
- Data analysis
- Sharing with healthcare providers
